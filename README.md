# Customer Value Identification & Segmentation System

I built this project to analyze customer financial behavior and identify which customers bring the most value to a business. Instead of simply comparing customers based on income or credit score, the system evaluates multiple financial factors and creates a Customer Value Index (CVI) to measure overall customer value.

The project uses statistical techniques such as Z-score normalization, ratio analysis, and customer segmentation to identify VIP customers, average customers, and high-risk customers. The results are visualized through an interactive Power BI dashboard that helps understand customer behavior, customer value distribution, and risk patterns.

This project demonstrates how data analytics can be used to support customer-focused business decisions in industries such as banking, lending, and financial services.
